# TMK Suite Website

🌐 [**tmk-tunnel.pl**](https://tmk-tunnel.pl) — oficjalna strona TMK Suite

Strona-wizytowka platformy do handlu algorytmicznego. Zawiera opisy wszystkich 5 aplikacji pipeline, galerie screenshotow, artykuly i system CMS.

## Tech stack

- **HTML/CSS/JS** — Single Page Application
- **Cloudflare Pages** — hosting + Functions for OG tags and SEO
- **Cloudflare Workers** — admin CMS panel
- **GitHub** — auto-deploy from main branch

## Pages

- `/` — Home with hero, app carousel, stats, tech stack
- `/projekt/[slug]` — Project detail pages
- `/artykul/[slug]` — Articles
- `/admin.html` — CMS panel
